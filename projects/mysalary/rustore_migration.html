Миграция на версию из RuStore

Если Вы пользователь из РФ, то скорее всего у Вас нет возможности оплачивать внутренние покупки через Google Play.

Приложение "Моя зарплата" опубликовано в магазине приложений RuStore, через который можно оплачивать покупки с использованием банковских карт, выпущенных в России или со счета мобильного телефона.

Приложение "Моя зарплата" с возможностью покупки платных функций опубликовано в магазине RuStore.

Оплачивать можно с использованием банковских карт, выпущенных в России или со счета мобильного телефона.

Ссылка: "Моя зарплата" в RuStore

Инструкция по миграции

Для перехода на версию из RuStore, необходимо выполнить несколько основных шагов:

1. Установка магазина приложений RuStore

  • перейти с мобильного телефона/планшета по ссылке "Моя зарплата" в RuStore

  • нажать кнопку "Установить с помощью RuStore" - отобразится инструкция по установке магазина

  • выполнить все шаги из инструкции Инструкция по установке RuStore

    • 2. Установка приложения "Моя зарплата"

  • зайти в приложение RuStore на устройстве, перейти в профиль и авторизоваться (необходимо для возможности совершать покупки)

  • скачать приложение "Моя зарплата", перейдя на мобильном устройстве по ссылке https://apps.rustore.ru/app/com.tadiuzzz.tadius.mysalary.rustore, либо введя в поиск в RuStore "Моя зарплата". Установится дублирующая версия - старая так же останется на телефоне.

  • перенести данные из старой версии, если необходимо (инструкция ниже)

  • после этого станет доступно совершение внутренних покупок расширенного функционала

    • 3. Перенос данных в новую версию

  • в старой версии зайти в настройки - резервное копирование - создать резервную копию - выбрать папку, куда сохранится копия - закрыть приложение

  • открыть новую версию - настройки - резервное копирование - восстановить резервную копию - открыть папку, в которую на предыдущем шаге сохранили копию - выбрать файл копии - закрыть приложение

  • удостовериться, что в обеих версиях приложения данные корректно отображаются и все успешно перенеслось

  • после этого можно удалить старую версию от Google Play, если она больше не нужна
