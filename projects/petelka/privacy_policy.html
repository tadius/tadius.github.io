Политика конфиденциальности мобильного приложения "Петелька"

Последнее обновление: 07.05.2025

Разработчик: Виталий Симонов

Настоящая Политика конфиденциальности описывает, как мобильное приложение "Петелька" (далее — "Приложение") собирает, использует и защищает ваши данные. Используя Приложение, вы соглашаетесь с условиями этой политики.

1. Сбор и использование данных

1.1. Обезличенная аналитика

Приложение собирает обезличенные данные о взаимодействии пользователей с интерфейсом для улучшения работы и устранения ошибок. Эти данные могут включать:

  • Типы выполняемых действий (например, нажатие кнопок, переходы между экранами)
  • Информацию о сбоях и ошибках
  • Данные о производительности устройства (скорость работы, использование памяти)

Эти данные не содержат личной информации и используются исключительно для отладки и поддержания работоспособности Приложения.

1.2. Распознавание голоса

Приложение использует стороннюю библиотеку для обработки голосовых команд. Ваши голосовые записи не передаются на сервер и обрабатываются локально на устройстве. Однако распознанный текст может передаваться в анонимной форме в аналитику для улучшения точности распознавания команд.

1.3. Реклама

Приложение использует SDK стороннего рекламного провайдера для показа рекламы. Рекламные сети могут собирать данные о ваших интересах для персонализации рекламы. Эти данные могут включать:

  • Идентификатор рекламного устройства (Advertising ID)
  • Данные о взаимодействии с рекламой

Мы не передаем рекламным сетям ваши персональные данные. Вы можете ограничить сбор рекламного идентификатора в настройках вашего устройства.

1.4. Платные подписки

Приложение предлагает платные подписки, которые обрабатываются через магазин приложений (RuStore / Google Play / App Store). Мы не получаем и не храним данные вашей платежной карты — вся обработка платежей осуществляется магазином приложений в соответствии с их политикой конфиденциальности.

2. Передача данных третьим лицам

Мы не передаем ваши личные данные третьим лицам, за исключением случаев:

  • Аналитических сервисов (в обезличенной форме)
  • Рекламных сетей (только Advertising ID и данные взаимодействия с рекламой)
  • Сторонних библиотек, необходимых для работы Приложения (например, распознавание голоса)

3. Безопасность данных

Мы принимаем разумные меры для защиты ваших данных, но не можем гарантировать абсолютную безопасность при передаче информации через интернет.

4. Изменения в политике конфиденциальности

Мы оставляем за собой право обновлять эту политику. При значительных изменениях уведомим вас через Приложение или по электронной почте.
